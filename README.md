TradeWise 🏦💹

TradeWise is a premium, interactive trading and finance platform designed for business owners, traders, and investors. It allows users to track profits, losses, expenses, and investments across multiple devices, offering real-time analytics and insights to make smarter financial decisions.

🌟 Features

📊 Real-Time Analytics – Monitor your profits, losses, and ROI instantly.

💼 Portfolio Management – Track multiple investments on one clean dashboard.

📱 Multi-Device Support – Works seamlessly on smartphones, tablets, and laptops.

🤖 Smart Insights – AI-powered suggestions for smarter trading decisions.

🔒 Secure Login & Signup – Protect your data while keeping access easy.

📅 Daily & Monthly Reports – Stay on top of your business growth.

🎨 Interactive & Premium UI – Modern design using #BE741E as the main accent color.

🚀 How TradeWise Works

Sign Up / Login 🔑

Users create an account or log in to access their dashboard.

The app ensures secure access and tracks users’ financial activity.

Dashboard Overview 🖥️📱

Users can view an interactive dashboard that displays their daily expenses, monthly profits, and investment summaries.

The dashboard is responsive and adapts to different devices like smartphones, tablets, and laptops.

Expense Tracking & Analysis 💸

Track income, expenses, and investments easily.

Graphs and charts highlight key insights with #BE741E as the main accent color, giving a professional and premium look.

Smart Recommendations 🤖

AI analyzes patterns in the user’s trading and spending.

Provides actionable suggestions for smarter investment decisions.

Interactive Reports & Alerts 📈

Users can generate daily, weekly, and monthly reports.

Alerts notify users about trends, losses, or opportunities for growth.

🛠️ Tech Stack

Frontend: React.js, Tailwind CSS

Backend:  Nest.js

Database: Prisma

Authentication: JWT / Session Management

Visualization: Chart.js / Recharts

Deployment: Vercel  (Frontend),  Render (Backend)

💻 Installation & Running Locally
# Clone the repository
git clone https://github.com/utujesandrine456/TradeWise.git

# Move into the project directory
cd TradeWise

# Install dependencies
npm install

# Start the frontend
npm run dev

# Start the backend
cd backend
npm install
npm run dev


Access the app at http://localhost:2016
