## 💡 About Stocka

> **Stocka** is a premium, interactive trading and finance platform designed for business owners, traders, and investors. Track profits, losses, expenses, and investments across multiple devices — with real-time analytics and AI-powered insights to help you make smarter financial decisions.

Whether you're monitoring daily expenses, managing a diverse portfolio, or seeking data-driven investment signals, Stocka delivers a clean, intuitive experience that scales with your financial goals.

## 🚀 How It Works

<br/>

### 1. 🔑 Sign Up / Login
Create your account or log in to securely access your personal dashboard. Stocka uses JWT authentication to protect your data while keeping the login experience smooth and intuitive.

<br/>

### 2. 🖥️ Dashboard Overview
View your interactive dashboard with a bird's-eye view of:
- Daily & monthly **expenses**
- **Profit/loss summaries**
- Investment breakdowns and portfolio snapshots

The dashboard is fully responsive — it adapts beautifully to any screen size.

<br/>

### 3. 💸 Expense Tracking & Analysis
Easily log and categorize your income, expenses, and investments. Dynamic charts and graphs — styled with the signature `#BE741E` accent — surface key insights at a glance, giving your workflow a polished, professional edge.

<br/>

### 4. 🤖 Smart AI Recommendations
Stocka's built-in AI analyzes patterns in your trading and spending behavior, delivering actionable suggestions to optimize your investment strategy over time.

<br/>

### 5. 📈 Reports & Alerts
Generate detailed financial reports on demand. Get notified about emerging trends, potential losses, and growth opportunities — so you're always one step ahead.

## 🏁 Getting Started

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) `v18+`
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [Git](https://git-scm.com/)

<br/>

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/utujesandrine456/Stocka.git

# 2. Move into the project directory
cd Stocka

# 3. Install frontend dependencies
npm install

# 4. Start the frontend dev server
npm run dev
```

```bash
# 5. In a separate terminal — start the backend
cd backend
npm install
npm run dev
```

<br/>

> 🌐 Access the app at **[http://localhost:2016](http://localhost:2016)**

<br/>

### Environment Variables

Create a `.env` file in the root directory and configure the following:

```env
# App
PORT=2016

# Database
DATABASE_URL="your_database_url"

# Authentication
JWT_SECRET="your_jwt_secret"
JWT_EXPIRES_IN="7d"
```
